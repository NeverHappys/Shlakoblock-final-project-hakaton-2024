Проект от команды Shlakoblock Inc представляет собой приложение для составления наборов блюд в условиях лечебных диет. Приложение принимает на вход от пользователя информацию о диете, которую ему назначил врач(15 столов по Певзнеру). Приложение на основе этой информации составляет рацион на неделю, расписанный по дням недели и приёмам пищи. Рекомендуется перед приготовлением блюд, предложенных нашим приложением проверять способ приготовления данного блюда, соответсвующий Вашей диете. Перед использованием приложения и переходом на диету проконсультируйтесь с врачом, какая именно диета Вам необходима.

Для запуска проекта понадбится установить 2 библиотеки:
1. flet "https://github.com/flet-dev/flet" or pip install flet in cmd
2. g4f "https://github.com/techwithanirudh/g4f" or install g4f in cmd
